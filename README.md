# MyEventLogViewer

このプロジェクトは、イベントログを閲覧・管理するアプリケーションです。

## プロジェクト構成

MyEventLogViewer/ ├── initiate.sh # 初期セットアップスクリプト ├── requirements.txt # 必要なPythonライブラリのリスト └── main.py # メインアプリケーションのコード


### initiate.sh
初回セットアップ用のスクリプトです。このスクリプトを実行することで、仮想環境の設定や必要な依存関係がインストールされます。
